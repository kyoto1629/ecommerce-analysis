Тема: Исследовательский анализ данных интернет-магазина: от EDA до сегментации клиентов и анализа воронок продаж.

О проекте:
В данном проекте проведен комплексный анализ поведения пользователей интернет-магазина.

Характеристики датасета:

Период: Октябрь — Ноябрь 2019
Пользователей: 1,160,808
Событий: ~2,200,000 (просмотры, добавления в корзину, покупки)
Категорий: 14 основных категорий, 15+ брендов
Ценовой диапазон: 0.81 — 2,574 руб.

Примечание о данных
Исходные файлы данных (`.csv`) не загружены в репозиторий из-за ограничений GitHub на размер файлов (лимит 100MB на файл).

Цель анализа:
Выявить паттерны поведения пользователей, оценить эффективность воронки продаж, сегментировать клиентскую базу и сформулировать рекомендации для увеличения конверсии и удержания.

Ключевые результаты:
Метрики воронки
View: 2,081,608 событий (100%)
Cart: 78,783 событий (3.78% от view)
Purchase: 33,291 событий (42.26% от cart)
Итоговая конверсия View → Purchase: 1.60%

RFM-сегментация:
Чемпионы: 17.5% (самые ценные, средний чек ~360 руб.)
В зоне риска: 22.4% (высокий исторический чек, требуют реактивации)
Потерянные: 27.0% (давно не покупали)
Новые: 14.4% (требуют онбординга)
Лояльные: 10.1%
Обычные: 8.6%

Удержание:
Retention Month 1: 29.6% (почти каждый 3-й возвращается)
Среднее число покупок: 1.10 на пользователя


Основные выводы:

Воронка продаж:
Сильная сторона: Cart -> Purchase (42.26%) — чекаут работает отлично.
Зона роста: View -> Cart (3.78%) — необходимо улучшить карточки товаров (добавить отзывы, триггеры срочности, рассрочку).

Категории:
Electronics — ядро бизнеса (2.48% конверсия). Масштабировать успешные практики.
Apparel — аутсайдер (0.48% конверсия). Требуется аудит: цены, контент, размерная сетка.
Unknown (32% событий) — критическая проблема качества данных. Необходима разметка.

Временные паттерны:
Пик: 16:00 UTC, пятница-суббота.
Рекомендация: Запускать промо-акции и рассылки в 14:00–15:00 UTC для максимального охвата.

Пользователи:
49.4% — "Потерянные" или "В зоне риска".
17.5% — "Чемпионы" (генерируют основную выручку).
Рекомендация: Сместить фокус с acquisition(привлечение новых пользователей) на retention(возврат старых пользователей). Реактивация 10% "В зоне риска" даст +2,200 покупателей.

Удержание:
Retention M1: 29.6% — хороший показатель, но есть куда расти.
Среднее число покупок: 1.10 — низкий repeat rate.
Рекомендация: Запустить post-purchase коммуникацию (скидка на 2-й заказ, например через 14 дней).

Визуализация и Дашборд
Скриншоты страниц интерактивного дашборда находятся в папке (dashboard/).

Рекомендации для бизнеса:
Приоритет 1: Улучшение View -> Cart
A/B тестирование элементов карточки товара
Добавление блока "С этим покупают"
Внедрение быстрых действий ("Купить в 1 клик")

Приоритет 2: CRM-стратегия
Для "В зоне риска": Персональные офферы ("Вернитесь, дарим 15%")
Для "Новых": Onboarding-цепочка (гайд + скидка)
Для "Чемпионов": Программа лояльности (кэшбэк, ранний доступ)

Приоритет 3: Качество данных
Разметка категории unknown (32% событий)
